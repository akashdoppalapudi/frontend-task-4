# Frontend-Task-4

This task is an Employee Directory built using HTML, CSS and JavaScript.

## Features

- Advanced Filtering options based on name, office, department and jobTitle
- Filter by Alphabet
- Instant Searching
- Can add Employees
- Can Edit details
- Can Delete Employees
